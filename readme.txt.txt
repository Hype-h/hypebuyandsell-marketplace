== Installation ==

1. Upload the plugin ZIP file via WordPress admin or extract to `/wp-content/plugins/`
2. Activate the plugin via the **Plugins** menu
3. Use the `[hbsvd_checkout]`, `[hbsvd_vendor_dashboard]`, etc., shortcodes in your pages
4. Configure vendor registration and dashboard links via WCFM settings
