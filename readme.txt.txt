== Description ==

**HypeBuy&Sell** is a fully functional multivendor marketplace plugin built for WooCommerce. Designed with performance and vendors in mind, it offers a custom checkout, vendor dashboards, analytics, shortcodes, Elementor sections, and more.

Perfect for building a premium online store where vendors can register, manage products, track orders, and monitor earnings.

= Features =
* 🛒 Custom styled checkout with trust icons
* 📦 Vendor dashboard with sales analytics (WCFM-compatible)
* 💬 Shortcodes for checkout, vendor panel, product grid, etc.
* 🎨 Elementor-ready home layout sections
* 📊 [hbsvd_analytics] for admin/vendor reports
* 🔐 Secure, mobile-first design
* 📁 Auto-page installer script
* 🧩 WordPress 6.5 & WooCommerce compatible

== Installation ==

1. Upload the plugin ZIP file via WordPress admin or extract to `/wp-content/plugins/`
2. Activate the plugin via the **Plugins** menu
3. Use the `[hbsvd_checkout]`, `[hbsvd_vendor_dashboard]`, etc., shortcodes in your pages
4. Configure vendor registration and dashboard links via WCFM settings

== Frequently Asked Questions ==

= Do I need WCFM installed? =
Yes. This plugin is compatible with WCFM for managing vendors and storefronts.

= Can vendors register for free? =
Yes! Use the built-in vendor form and connect it to WCFM’s capabilities.

= Does it support Elementor? =
Yes. You’ll find sections built for Elementor like hero blocks, vendor grids, and stats.
